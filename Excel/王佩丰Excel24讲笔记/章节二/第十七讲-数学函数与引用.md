# 1、ROUNDUP函数，无条件向上进位。如ROUNDUP(A1,2)，则保留两位小数，如果后面仍有小数，则进位



# 2、ROUNDDOWN函数，无条件向下舍位。如ROUNDDOWN(A1,0)，则保留整数、舍弃小数



# 3、INT函数，直接取整、保留整数



# 4、MOD函数，求余数，该函数常用于判断一个数是奇数还是偶数
